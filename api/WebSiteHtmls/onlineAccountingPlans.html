اکانتینگ آنلاین هات اسپات پلاس یک سرویس ابری است. این سرویس به مدیران شبکه این امکان را می دهد که با استفاده از یک روتر میکروتیک مصرف اینترنت کاربران خود را کنترل کنند. این کنترل با راه اندازی یک هات اسپات و تعریف بسته های اینترنت، برای مدیریت حجم و سرعت اتصال کاربران به اینترنت است.

 سرویس اقتصادی

این سرویس برای کافه‌ها، رستوران‌ها، و هتل‌های کوچک مناسب است و اتصال کاربران را تا ۱۲۰ کلاینت همزمان ضمانت می‌کند.
